Materialize & ASP.net

Responsive and Modern Frameworks

Materialize

Materialize CSS is a UI component library which is created with CSS, JavaScript and HTML. It is created and designed by Google. Materialize CSS is also known as Material Design. It is a design language which combines the classic principles of successful design along with innovation and technology. Google's goal is to develop a system of design that allows for a unified user experience across all their products on any platform. It is used to construct attractive, consistent, and functional web pages and web apps while adhering to modern web design principles such as browser portability, device independence, and graceful degradation.

Popularity

Materialize implement Google's Material Design, and is thus one of the most popular design languages out there. Google has done an incredible job with pushing this design language and has created a very consistent and elegant look and feel on the Android platform. Materialize is ideal for modern web and app design.

Strengths

  • • It is a standard CSS with minimal footprint.

  • • Built-in responsive design

  • • It is free to use and requires jQuery JavaScript library to function properly.

  • • Customizable

  • • It is cross-browser, compatible, and can be used to create reusable web components.

  • • It provides new versions of common user interface controls such as buttons, checkboxes, and text fields adapted to follow Material Design concepts.

  • • Large selection of components

Weaknesses

  • • Refuses to use the flexbox mode

  • • Large / heavy


ASP.net

ASP.NET Core is an open source web framework developed by Microsoft for creating dynamic websites and applications. Initially released in 2016, It was designed to be a more community focused version of Microsoft's original ASP.NET framework. ASP.NET Core was more modular and lightweight compared to the ASP.NET Framework. It blends traditional HTML/CSS/Javascript web development with Microsoft's Visual Basic and C# and F# programming languages. Code can be written in other languages like C++ as well.

Popularity

Developers often pick ASP.NET Core for its integration with the Microsoft Ecosystem, scalability and performance. It is popular with Enterprise level development for its abundance of features and focus on security. Also its use of the MVC (Model-View-Controller) architectural pattern allows for multiple collaborators to contribute on a project simultaneously, since elements of the project are isolated from one another. ASP.NET Core uses the .NET Core executable framework. In Stack Overflow’s annual developer survey, .NET Core was cited as being the third “Most Popular Framework” (23.7%) behind .NET (37.4%) and Node.js (49.9%). Despite being the third most popular, it polled first in the category of “Most Loved Frameworks”.

Strengths

  • • Open source

  • • Modular

  • • Benefits Collaborative projects due to MVC pattern

  • • Razor Core provides a very seamless integration of C# code into traditional HTML pages

  • • Feature rich

  • • Visual Studio IDE provides really good debugging, intellisense and lots of code templates for ASP.NET Core

  • • Common web development tasks have simple built-in methods that can be called

Weaknesses

  • • Amount of features may be overwhelming

  • • Steep learning curve

  • • Despite the framework being cross platform, many of its development tools are either exclusive to windows, or lack features on other platforms

  • • A lot of the advantages of ASP.NET Core come with its implementation in the Visual Studio IDE so if you prefer using a text editor like vim or notepad++ you may end up having to do a lot more work even with plugins installed
